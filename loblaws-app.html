Loblaws App Concept

Map My Shop

Type
UI/UX, Group project
Role
lorem ipsum
Tools
Adobe XD
Date
2019

Outline

The goal of this project was to create an app concept for Loblaws that utilized both 5G and AR.

Focusing on target market research led us to the understanding that many people dislike grocery shopping, but don't necessarily trust a shopping service to shop for them.

Die Lines

Using Illustrator, I created die lines for the new package which would fit on a single sheet of letter sized paper using only a single glue line, and fold lines for the rest. An optional clear sticker on the top and bottom of the package may be used to prevent the package being opened in-store.

Typography

By carefully selecting a new font family, I was able to include all of the same information, while making the package more readable, and decrease the overall width of the backing.
